# Selten WorkFlow

Task management app with Kanban board, sprint planning, and history tracking.

## Getting started

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` - start dev server
- `npm run build` - production build
- `npm run preview` - preview production build
- `npm run lint` - run eslint

## Tech

- Vite + React
- Tailwind CSS + shadcn/ui
